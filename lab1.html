Lab 1: The Artemis Board

The purpose of this lab is to setup and become familiar with the Arduino IDE and the Artemis board.

In the first lab, we set up the Artemis board on our computer and tested the connection using the four example codes.

Part 1: Blink it Up!

After the Arduino is connected and ready to go, we used the default "Blink" code in Arduino IDE. The video of the Arduino blinking is shown below!

Part 2: Serial Reader

Here, the serial connection between the Artemis board and the computer is tested. The image below shows the output of the code Example2_Serial given in the Artemis example codes.

Part 3: Temperature Sensor

Using Example4_analogRead code in the Artemis example codes, we tested the temperature sensor on the Arduino by tracking the temp (counts) on the Serial Monitor. As I held onto the Artemis board, the temp (counts) increases from around 33500 until about 33700.

Part 4: Microphone

This time, we tested the microphone by printing the loudest frequency on the serial monitor using the Example1_MicrophoneOutput code. When I whistled near the board, the loudest frequency magnitude increased from about 300s until 1500s.
